# üßÆ 4Ô∏è‚É£ SUBNETTING CALCULATION FORMULA

### üß© Formula 1: Number of Subnets

```
Number of Subnets = 2^(new prefix - old prefix)
```

### üß© Formula 2: Hosts per Subnet

```
Hosts/Subnet = 2^(32 - new prefix)
Usable hosts = Hosts/Subnet - 2
```

---

### Example:

Base CIDR: `10.0.0.0/16`
New Subnet: `/24`

1Ô∏è‚É£ **Subnets ‡¶§‡ßà‡¶∞‡¶ø ‡¶π‡¶¨‡ßá:**
`2^(24 - 16) = 2^8 = 256 subnets`

2Ô∏è‚É£ **‡¶™‡ßç‡¶∞‡¶§‡¶ø‡¶ü‡¶ø Subnet ‡¶è usable host:**
`2^(32 - 24) - 2 = 254 hosts`
