what is DevOps(principle and role) ?

stapes:
    1. plan: requirement analysis ,work place, server configuration, soft architecture, system design
    2. code: feature development
    3. build: test properly deploy and run application, anywhere run
    4. test: tdd, etc
    5. release: release stable code, artifact: package or docker image
    6. deploy : live
    7. operate:
    8. monitor: debugging and Logging


Software Development Life Cycle:
    stage 1: Planning & requirement analysis
    stage 2: Defining requirements
    stage 3: Architecture Design
    stage 4: Software Development
    stage 5: Testing
    stage 6: Deployment and Maintenance


Agile Methodology:
    1. Plane
    2. Design
    3. Develop
    4. Test
    5. Deploy
    6. Review
    7. Launch


Devops(secure and govern):
    1. Plane
    2. create
    3. verify
    4. package
    5. deploy
    6. monitor: resource consume, cpu, ram
    7. repeat

# devops secure and governed
# scale up

Common roles and responsibilities:
    1. Application & infrastructure planning, testing and development
    2. Maintaining CI/CD pipeline
    3. Automation implementation
    4. On-call incident response and incident management
    5. monitoring
